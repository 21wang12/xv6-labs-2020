# xv6-labs-2020
MIT 6.s081 lab repo for 2020 fall

+ Lab 1 is in __`util`__ branch
+ Lab 2 is in __`syscall`__ branch
+ Lab 3 is in __`pgtbl`__ branch

please run `git checkout [branch]` to switch into different lab. for example:
```bash
# change to util lab
cd xv6-labs-2020
git checkout util
```
